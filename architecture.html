What the Server Sees vs What the Client SeesZERO-KNOWLEDGE

🖥️ Server (encrypted blobs only)

SRP Verifier
Not reversible to password
Encrypted Private Key
AES-256-GCM blob — unreadable
Encrypted Vault Keys
Encrypted with user's pub key
Encrypted Items
Opaque blobs — server cannot read contents
Public Key (plaintext)
Safe to share — compromises nothing
Salts + KDF Params
Public by design — no compromise

💻 Client (everything in plaintext, in memory)

Master Password
Memory only, cleared after KDF
Secret Key (128-bit)
Device only, never on server
Encryption Key + Auth Key
Derived, in memory (chrome.storage.session)
Vault Keys (decrypted)
In memory, auto-lock 15 min
Plaintext Passwords
Decrypted on-demand, cleared after use
Private Key (decrypted)
For vault sharing (crypto_box)
